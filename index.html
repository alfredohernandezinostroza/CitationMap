Citation Network Analysis Tool

This tool allows you to analyze citation networks by uploading a network file (GEXF or GraphML). You can visualize the network interactively, explore paper metadata, filter by various criteria, and view top papers and authors by subfield.

Upload Network File

How to use this tool

  1. Upload a network file (GEXF, GraphML, or XML) using the file uploader above
  2. Explore the network visualization by zooming, panning, and clicking on nodes
  3. Filter the network by cluster, year, or author
  4. View paper details for each node in the network
  5. Navigate to the Top Rankings page to see top papers and authors by subfield
About network file formats
  • GEXF (Graph Exchange XML Format) is an XML-based format for describing network structures.
  • GraphML is another XML-based file format for graphs.

Both formats represent papers as nodes and citations as edges. The application will attempt to determine the correct format automatically.
